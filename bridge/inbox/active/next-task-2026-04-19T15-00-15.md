Compatibility API 백엔드 함수 구현 (Prokerala Kundli Matching)

## Instructions
## 목표
`functions/ai/birth-chart.js` 구조를 참고하여 Prokerala Kundli Matching(궁합) API를 호출하는 백엔드 Cloud Function을 구현한다.

## 구현 사항

### 1. `functions/ai/compatibility.js` 신규 생성
- `functions/utils/prokerala-auth.js`의 `getProkeralaToken()` 재사용
- Prokerala API 엔드포인트: `https://api.prokerala.com/v2/astrology/kundli-matching`
- 요청 파라미터:
  - `girl_dob` (ISO 8601, e.g. `1995-06-15T10:30:00+05:30`)
  - `girl_coordinates` (`lat,lon` 문자열)
  - `boy_dob`
  - `boy_coordinates`
  - `ayanamsa` (기본값: `1` = Lahiri)
- 응답에서 다음 필드만 추출하여 반환:
  ```json
  {
    "total_points": 28,
    "max_points": 36,
    "compatibility_percentage": 77.8,
    "kuta_details": [...],
    "mangal_dosha": { "girl": false, "boy": false }
  }
  ```
- 에러 처리: 401(토큰 만료) → 토큰 재발급 후 1회 재시도, 그 외 에러는 `{ error: message }` 반환
- `exports.getCompatibility` 로 export

### 2. `functions/index.js` 수정
- 기존 `birth-chart` import 패턴과 동일하게 2줄 추가:
  ```js
  const { getCompatibility } = require('./ai/compatibility');
  exports.getCompatibility = getCompatibility;
  ```
