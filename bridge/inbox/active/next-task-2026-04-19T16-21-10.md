## Instructions
## 목표
Prokerala API를 사용하여 출생 차트(Birth Chart) 데이터를 가져오는 백엔드 Cloud Function을 구현한다.

## 작업 범위

### 1. 환경 변수 확인
`functions/.env` 또는 Firebase 환경 설정에 다음 키가 존재하는지 확인:
- `PROKERALA_CLIENT_ID`
- `PROKERALA_CLIENT_SECRET`

없으면 `functions/.env.example`에 플레이스홀더를 추가하고, 실제 값은 **절대 코드에 하드코딩하지 말 것**.

### 2. Prokerala OAuth 토큰 헬퍼 생성
`functions/ai/prokerala/auth.js` 파일 생성:
```js
// Prokerala OAuth2 client_credentials 방식 토큰 발급 및 캐싱
// 토큰 만료 전 자동 갱신 (expires_in 기반)
// 메모리 캐시 사용 (Cloud Function 인스턴스 수명 내)
```

### 3. Birth Chart API 래퍼 생성
`functions/ai/prokerala/birthChart.js` 파일 생성:
- Endpoint: `https://api.prokerala.com/v2/astrology/birth-chart`
- 필수 파라미터: `datetime`, `coordinates`, `ayanamsa`(기본값: `1` = Lahiri)
- 응답 데이터를 앱에서 사용하기 쉬운 형태로 정규화하는 `normalizeBirthChart(raw)` 함수 포함
- 에러 처리: HTTP 4xx/5xx 시 명확한 에러 메시지 반환

### 4. Firebase Callable Function 생성
`functions/ai/prokerala/getBirthChart.js` 파일 생성:

### 5. functions/index.js 등록
`functions/index.js`에 `getBirthChart` export 추가.

### 6. 단위 테스트

- Prokerala API mock 처리 (실제 API 호출 금지)
- 정상 응답 정규화 테스트
- 잘못된 입력 에러 처리 테스트
- 토큰 캐싱 동작 테스트
